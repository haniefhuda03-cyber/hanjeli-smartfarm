# Suggested Commands for Development

* **Start Development Server**: `npm run dev` (Starts Next.js app)
* **Build for Production**: `npm run build`
* **Start Production Server**: `npm run start`
* **Lint Code**: `npm run lint`

**Project Details**:
* Project: Hanjeli Smart Farm
* Tech Stack: Next.js (App Router), TypeScript, Tailwind CSS, Lucide Icons, Recharts, Sonner (for toast notifications), React.
* It is a React frontend application targeting a Smart Farming system.

**Workflow**:
Since the user mentioned this is a front-end only implementation simulation, no backend is present, we will rely on UI-side code, logic with simple timeout delays and mock data representations.
